# PromptForge — AI Prompt Optimizer

Paste any prompt and get instant AI-powered analysis, scoring, and rewrites.

## Features
- **5-dimension scoring** — Clarity, Specificity, Context, Instructions, Output Format
- **3 rewrite variants** — Minimal, Structured, Expert
- **Live comparison** — Run original vs rewritten prompt side by side
- **Instant feedback** — Know exactly what's weak and why

## Stack
- FastAPI + Groq API (Llama 3.3 70B)
- Deployed on HuggingFace Spaces (Docker)
- Frontend: GitHub Pages
